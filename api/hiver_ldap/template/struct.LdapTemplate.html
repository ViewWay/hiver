Struct LdapTemplate 

Source 
pub struct LdapTemplate { /* private fields */ }
Expand description

Central class for LDAP operations, wrapping a ContextSource. LDAP操作的中心类，包装 ContextSource。

§Example / 示例

ⓘ
use hiver_ldap::{LdapTemplate, LdapContextSource};

let context_source = LdapContextSource::new("ldap://localhost:389", "dc=example,dc=com");
let template = LdapTemplate::new(context_source);
