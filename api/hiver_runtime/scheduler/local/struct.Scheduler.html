Local scheduler for a single thread 单线程的本地调度器

Each scheduler runs on its own thread and manages its own task queue. Each scheduler follows the thread-per-core model with no work stealing.

每个调度器在自己的线程上运行并管理自己的任务队列。 每个调度器遵循 thread-per-core 模型，没有工作窃取。
