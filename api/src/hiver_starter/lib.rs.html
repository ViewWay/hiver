2//! Hiver Starter - Spring Boot风格的自动配置
3//! Hiver Starter - Spring Boot-style Auto-Configuration
4//!
5//! 这个 crate 提供了类似 Spring Boot Starter 的自动配置能力。
6//! This crate provides Spring Boot Starter-like auto-configuration capabilities.
7//!
8//! # 功能特性 / Features
9//!
10//! - **自动配置** - 根据类路径和配置自动装配组件
11//! - **条件装配** - 基于条件注解的智能装配
12//! - **依赖注入** - IoC 容器和依赖注入
13//! - **组件扫描** - 自动发现和注册组件
14//! - **配置管理** - 多层配置加载和覆盖
15//!
16//! # 使用示例 / Usage Examples
17//!
18//! ## 最简单的应用 / Minimal Application
19//!
20//! ```rust,no_run,ignore
21//! use hiver_starter::prelude::*;
22//!
23//! #[hiver_main]
24//! struct MyApp;
25//!
26//! fn main() -> Result<(), Box<dyn std::error::Error>> {
27//!     MyApp::run()
28//! }
29//! ```
30//!
31//! ## 完整应用 / Full Application
32//!
33//! ```rust,no_run,ignore
34//! use hiver_starter::prelude::*;
35//!
36//! #[hiver_main]
37//! #[component_scan]
38//! struct Application;
39//!
40//! #[controller]
41//! struct HelloController;
42//!
43//! #[get("/")]
44//! fn hello() -> &'static str {
45//!     "Hello, Hiver!"
46//! }
47//! ```
