# hiver-grpc

**gRPC integration for the Hiver Framework.**

**Hiver框架的 gRPC 集成。**

## Overview / 概述

`hiver-grpc` provides gRPC server and client support with protobuf service definitions and streaming.

`hiver-grpc` 提供 gRPC 服务器和客户端支持，支持 Protobuf 服务定义和流式传输。

## Features / 功能

See [API Documentation](https://docs.rs/hiver-grpc) for details.

详见 [API 文档](https://docs.rs/hiver-grpc)。

## Installation / 安装

```toml
[dependencies]
hiver-grpc = { version = "0.1" }
```
