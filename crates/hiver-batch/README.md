# hiver-batch

**Batch processing framework for the Hiver Framework (equivalent to Spring Batch).**

**Hiver框架的批处理框架（等价于 Spring Batch）。**

## Overview / 概述

`hiver-batch` provides batch processing abstractions for chunk-oriented processing with readers, processors, and writers.

`hiver-batch` 提供批处理抽象，支持面向块的读取、处理和写入。

## Features / 功能

See [API Documentation](https://docs.rs/hiver-batch) for details.

详见 [API 文档](https://docs.rs/hiver-batch)。

## Installation / 安装

```toml
[dependencies]
hiver-batch = { version = "0.1" }
```
