# hiver-ai

**AI integration for the Hiver Framework.**

**Hiver框架的 AI 集成。**

## Overview / 概述

`hiver-ai` provides unified AI model integration supporting OpenAI, Anthropic, and local models via Ollama.

`hiver-ai` 提供统一的 AI 模型集成，支持 OpenAI、Anthropic 和通过 Ollama 的本地模型。

## Features / 功能

See [API Documentation](https://docs.rs/hiver-ai) for details.

详见 [API 文档](https://docs.rs/hiver-ai)。

## Installation / 安装

```toml
[dependencies]
hiver-ai = { version = "0.1" }
```
