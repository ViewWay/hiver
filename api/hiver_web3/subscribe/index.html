WebSocket subscription module WebSocket订阅模块

§Overview / 概述

This module provides WebSocket-based event subscriptions for blockchain events. It supports subscribing to new blocks, pending transactions, and contract events.

本模块提供基于WebSocket的区块链事件订阅功能。 支持订阅新区块、待处理交易和合约事件。

§Equivalent to Spring Boot / 等价于 Spring Boot

  • Spring WebSocket (WebSocketStompClient)
  • Web3j WebSocket subscriptions
  • Spring WebFlux event streaming

§Example / 示例

ⓘ
use hiver_web3::subscribe::{WsClient, SubscriptionType};
use futures_util::StreamExt;

let client = WsClient::connect("wss://eth.llamarpc.com").await?;

// Subscribe to new blocks
let mut blocks = client.subscribe_blocks().await?;

while let Some(block) = blocks.next().await {
    println!("New block: {:?}", block);
}
