LDAP template pattern — simplified LDAP operations LDAP模板模式 — 简化的LDAP操作

Equivalent to Spring LDAP’s LdapTemplate. When the ldap feature is enabled, operations connect to a real LDAP server. Otherwise they return safe defaults for API compatibility testing.

等价于 Spring LDAP 的 LdapTemplate。 启用 ldap feature 时，操作会连接到真实的LDAP服务器。 否则返回安全的默认值以支持API兼容性测试。
