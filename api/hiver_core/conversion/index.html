Type conversion service 类型转换服务

Equivalent to Spring’s ConversionService. 等价于 Spring 的 ConversionService。

Provides a pluggable type conversion pipeline for converting between types. Commonly used for property binding, request parameter parsing, and @Value injection.

提供可插拔的类型转换管道，用于在类型之间转换。 常用于属性绑定、请求参数解析和 @Value 注入。
