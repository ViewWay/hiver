Security filter chain 安全过滤器链

Equivalent to Spring Security’s SecurityFilterChain. 等价于 Spring Security 的 SecurityFilterChain。

Provides a composable chain of security filters that process requests in order: authentication, authorization, CSRF, CORS, etc.

提供可组合的安全过滤器链，按顺序处理请求： 认证、授权、CSRF、CORS 等。
