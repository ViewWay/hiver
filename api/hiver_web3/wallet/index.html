Wallet module 钱包模块

§Overview / 概述

This module provides wallet management including address handling, transaction signing, and key management.

本模块提供钱包管理，包括地址处理、交易签名和密钥管理。

§Equivalent to Spring Boot / 等价于 Spring Boot

  • Spring Web3j (Wallet, Credentials)
  • Web3j Wallet management

§Example / 示例

ⓘ
use hiver_web3::wallet::{LocalWallet, Wallet, Address};

// Create a random wallet
let wallet = LocalWallet::random();
println!("Address: {}", wallet.address());

// Create from private key
let wallet = LocalWallet::from_private_key("0x...")?;
