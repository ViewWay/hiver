# hiver-cli

**Hiver project scaffolding CLI tool.**

**Hiver 项目脚手架工具。**

## Overview / 概述

`hiver-cli` provides a command-line tool for creating and managing Hiver projects, generating code scaffolding and configuration files.

`hiver-cli` 提供命令行工具，用于创建和管理 Hiver 项目，生成代码脚手架和配置文件。

## Features / 功能

See [API Documentation](https://docs.rs/hiver-cli) for details.

详见 [API 文档](https://docs.rs/hiver-cli)。

## Installation / 安装

```toml
[dependencies]
hiver-cli = { version = "0.1" }
```
