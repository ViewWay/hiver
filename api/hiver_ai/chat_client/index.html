Chat client with fluent API for building and sending AI requests. 用于构建和发送 AI 请求的流畅 API 聊天客户端。

The ChatClient provides a high-level, builder-style interface for interacting with chat models. It simplifies common patterns like setting system prompts, adding user messages, and calling the model.

ChatClient 提供高级的 builder 风格接口用于与聊天模型交互。 它简化了设置系统提示、添加用户消息和调用模型等常见模式。

§Example / 示例

ⓘ
use hiver_ai::chat_client::ChatClient;

let response = client
    .prompt("Translate to French: Hello")
    .system("You are a translator.")
    .call()
    .await?;
