# RDBC - Database Connectivity / RDBC 数据库连接

> **Status**: 🔄 In Development
> **状态**: 开发中

---

## Overview / 概述

Nexus RDBC provides a unified database client abstraction with connection pooling, similar to JDBC in the Java ecosystem.
