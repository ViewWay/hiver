1//! Hiver Router - HTTP request router
2//! Hiver路由器 - HTTP请求路由器
3//!
4//! # Overview / 概述
5//!
6//! `hiver-router` provides efficient HTTP request routing with path parameters
7//! and middleware support.
8//!
9//! `hiver-router` 提供高效的HTTP请求路由，支持路径参数和中间件。
10//!
11//! # Equivalent to Spring Boot / 等价于 Spring Boot
12//!
13//! - @`RequestMapping`, @`GetMapping`, @`PostMapping`, etc.
14//! - `PathVariable` extraction
15//! - `RequestParam` extraction
16//!
17//! # Example / 示例
18//!
19//! ```rust,no_run,ignore
20//! use hiver_router::Router;
21//! use hiver_http::{Method, Response, StatusCode};
22//!
23//! let router = Router::new()
24//!     .get("/users/:id", get_user)
25//!     .post("/users", create_user)
26//!     .route("/users/:id/posts", Method::GET, list_user_posts);
27//!
28//! async fn get_user(id: u64) -> Response {
29//!     Response::builder()
30//!         .status(StatusCode::OK)
31//!         .body(format!("User {}", id))
32//!         .unwrap()
33//! }
34//! ```
