Convenience function: execute a future with a timeout. 便捷函数：在超时限制下执行 future。

This is a lightweight helper that does not track metrics or invoke callbacks. For full-featured timeout management use Timeout::call.

这是一个轻量辅助函数，不跟踪指标也不调用回调。 如需完整功能，请使用 Timeout::call。
