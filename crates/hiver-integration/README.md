# hiver-integration

**Integration patterns for the Hiver Framework.**

**Hiver框架的集成模式。**

## Overview / 概述

`hiver-integration` provides enterprise integration patterns including message channels, transformers, and routing.

`hiver-integration` 提供企业集成模式，包括消息通道、转换器和路由。

## Features / 功能

See [API Documentation](https://docs.rs/hiver-integration) for details.

详见 [API 文档](https://docs.rs/hiver-integration)。

## Installation / 安装

```toml
[dependencies]
hiver-integration = { version = "0.1" }
```
