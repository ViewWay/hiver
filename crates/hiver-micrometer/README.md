# hiver-micrometer

**Metrics collection for the Hiver Framework (equivalent to Micrometer).**

**Hiver框架的指标收集（等价于 Micrometer）。**

## Overview / 概述

`hiver-micrometer` provides metrics collection and export with counters, gauges, histograms, and Prometheus integration.

`hiver-micrometer` 提供指标收集和导出，支持计数器、仪表、直方图和 Prometheus 集成。

## Features / 功能

See [API Documentation](https://docs.rs/hiver-micrometer) for details.

详见 [API 文档](https://docs.rs/hiver-micrometer)。

## Installation / 安装

```toml
[dependencies]
hiver-micrometer = { version = "0.1" }
```
