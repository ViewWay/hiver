# hiver-spel

**Spring Expression Language (SpEL) equivalent for Hiver.**

**Hiver 的表达式语言（SpEL 等价物）。**

## Overview / 概述

`hiver-spel` provides an expression language engine for runtime evaluation of expressions in configuration and annotations.

`hiver-spel` 提供表达式语言引擎，用于运行时评估配置和注解中的表达式。

## Features / 功能

See [API Documentation](https://docs.rs/hiver-spel) for details.

详见 [API 文档](https://docs.rs/hiver-spel)。

## Installation / 安装

```toml
[dependencies]
hiver-spel = { version = "0.1" }
```
