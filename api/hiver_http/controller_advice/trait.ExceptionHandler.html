Expand description

Trait for exception handlers that can process std::error::Error values. 可处理 std::error::Error 值的异常处理器 Trait。

Each handler declares whether it can handle a given error via can_handle, then produces an ErrorResponse via handle.

等价于 Spring 的 @ExceptionHandler 方法。
