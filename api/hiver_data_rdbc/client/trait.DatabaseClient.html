Database client trait 数据库客户端 trait

fn current_transaction(&self) -> Option<Transaction>

Get the current transaction from the active transaction context, if any. 从活跃事务上下文中获取当前事务（如果有）。

When the tx-bridge feature is enabled and a transaction has been started via RdbcTransactionManager, this method returns it.

当启用 tx-bridge feature 且通过 RdbcTransactionManager 启动了事务时， 此方法返回该事务。
