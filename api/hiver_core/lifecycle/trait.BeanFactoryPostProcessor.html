Expand description

Hook for custom modification of the bean factory’s definition registry. 用于自定义修改Bean工厂定义注册表的钩子。

Equivalent to Spring’s BeanFactoryPostProcessor. 等价于 Spring 的 BeanFactoryPostProcessor。

Called once during context startup, before any beans are instantiated. 在上下文启动期间，在实例化任何Bean之前调用一次。
