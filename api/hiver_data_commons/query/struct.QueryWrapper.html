Expand description

Query wrapper for building queries 查询包装器，用于构建查询

This is similar to MyBatis-Plus QueryWrapper, providing a fluent API for building query conditions.

这类似 MyBatis-Plus QueryWrapper，提供流畅的 API 用于构建查询条件。

§Example / 示例

ⓘ
use hiver_data_commons::QueryWrapper;

let query = QueryWrapper::new()
    .eq("status", "active")
    .ge("age", 18)
    .like("name", "Alice")
    .in_("city", vec!["Beijing", "Shanghai"])
    .order_by_asc("created_at");
