Struct TrieRouter 

Source 
pub struct TrieRouter { /* private fields */ }
Expand description

Trie-based router with efficient path parameter extraction 基于 Trie 的高效路径参数提取路由器

This is equivalent to Spring’s AntPathMatcher combined with @RequestMapping for path pattern matching.

这等价于Spring的AntPathMatcher结合@RequestMapping进行路径模式匹配。
