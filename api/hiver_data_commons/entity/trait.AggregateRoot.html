Aggregate root trait 聚合根 trait

Base trait for all entities in the domain model. 领域模型中所有实体的基础 trait。

§Example / 示例

ⓘ
use hiver_data_commons::AggregateRoot;

#[derive(Debug, Clone)]
struct User {
    id: i32,
    name: String,
}

impl AggregateRoot for User {
    type Id = i32;

    fn id(&self) -> &Self::Id {
        &self.id
    }

    fn set_id(&mut self, id: Self::Id) {
        self.id = id;
    }
}
