# hiver-events-macros

**Procedural macros for the Hiver event system.**

**Hiver事件系统的过程宏。**

## Overview / 概述

`hiver-events-macros` provides procedural macros for event handling, including `#[event_handler]` and event derive macros.

`hiver-events-macros` 提供事件处理的过程宏，包括 `#[event_handler]` 和事件派生宏。

## Features / 功能

See [API Documentation](https://docs.rs/hiver-events-macros) for details.

详见 [API 文档](https://docs.rs/hiver-events-macros)。

## Installation / 安装

```toml
[dependencies]
hiver-events-macros = { version = "0.1" }
```
