# hiver-shell

**Interactive shell (REPL) support for Hiver Framework**

**Hiver框架的交互式 Shell (REPL) 支持**

## Overview / 概述

`hiver-shell` provides an interactive shell framework for building CLI applications with command registration, help generation, and tab completion.

`hiver-shell` 提供交互式 Shell 框架，用于构建带有命令注册、帮助生成和 Tab 补全的 CLI 应用。

## Features / 功能

- **Command Registration** - Declarative command definitions
- **Help System** - Auto-generated help and usage text
- **Tab Completion** - Smart completion support
- **History** - Command history with persistence
- **Script Execution** - Batch command execution

- **命令注册** - 声明式命令定义
- **帮助系统** - 自动生成帮助和使用说明
- **Tab 补全** - 智能补全支持
- **历史记录** - 带持久化的命令历史
- **脚本执行** - 批量命令执行
