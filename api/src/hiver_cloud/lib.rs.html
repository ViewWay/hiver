1//! Hiver Cloud - Spring Cloud equivalent features
2//! Hiver云 - Spring Cloud等价功能
3//!
4//! # Equivalent to Spring Cloud / 等价于 Spring Cloud
5//!
6//! - `@EnableDiscoveryClient` - Service Discovery
7//! - `@EnableConfigServer` - Config Server
8//! - `@EnableZuulProxy` / `@EnableGateway` - Gateway
9//! - `@EnableCircuitBreaker` - Circuit Breaker (Resilience4j)
10//! - `@EnableRetry` - Retry
11//! - `@EnableFeignClients` - HTTP Clients (Feign)
12//!
13//! # Modules / 模块
14//!
15//! - `discovery` - Service discovery (Eureka, Consul, etcd)
16//! - `config` - Distributed configuration (Spring Cloud Config)
17//! - `config_client` - Enhanced config client with composite sources
18//! - `gateway` - API Gateway (Spring Cloud Gateway)
19//! - `circuit_breaker` - Circuit breaker pattern
20//! - `load_balancer` - Client-side load balancing
21//! - `http_client` - Declarative HTTP clients (Feign equivalent)
