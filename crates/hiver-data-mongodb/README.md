# hiver-data-mongodb

**MongoDB integration for the Hiver Framework (Repository, ReactiveMongoTemplate).**

**Hiver框架的 MongoDB 集成（Repository、ReactiveMongoTemplate）。**

## Overview / 概述

`hiver-data-mongodb` provides MongoDB integration with Repository traits, document mapping, and reactive template operations.

`hiver-data-mongodb` 提供 MongoDB 集成，支持 Repository trait、文档映射和响应式模板操作。

## Features / 功能

See [API Documentation](https://docs.rs/hiver-data-mongodb) for details.

详见 [API 文档](https://docs.rs/hiver-data-mongodb)。

## Installation / 安装

```toml
[dependencies]
hiver-data-mongodb = { version = "0.1" }
```
