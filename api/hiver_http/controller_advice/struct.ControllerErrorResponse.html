Struct ControllerErrorResponse

pub struct ControllerErrorResponse {
    pub status: u16,
    pub error: String,
    pub message: String,
    pub path: Option<String>,
    pub timestamp: u64,
}
Expand description

Standard error response produced by the controller advice. 控制器通知产生的标准错误响应。

JSON format:

{
  "status": 404,
  "error": "Not Found",
  "message": "User with id 123 not found",
  "path": "/api/users/123",
  "timestamp": 1706500000
}

§error: String

HTTP reason phrase (e.g. “Not Found”) HTTP 原因短语（如 “Not Found”）

Human-readable error message 人类可读的错误消息

§path: Option<String>

Request path that triggered the error 触发错误的请求路径

§timestamp: u64

Unix timestamp (seconds since epoch) Unix 时间戳（自纪元以来的秒数）
