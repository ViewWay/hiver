Trait for events that can provide property values by path 可以按路径提供属性值的事件trait

Events implementing this trait can be filtered using PropertyCondition. The property path uses dot notation (e.g., “user.name”).

实现此trait的事件可以使用 PropertyCondition 进行过滤。 属性路径使用点号表示法（例如 “user.name”）。
