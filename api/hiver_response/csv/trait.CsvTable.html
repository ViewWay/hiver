Expand description

Trait for types that can be auto-exported to CSV. 可自动导出到 CSV 的类型的 trait。

Equivalent to Spring Batch’s FieldSetMapper or OpenCSV’s bean mapping. 等价于 Spring Batch 的 FieldSetMapper 或 OpenCSV 的 Bean 映射。

§Example / 示例

ⓘ
use hiver_response::csv::{CsvTable, export_to_csv};

struct User {
    name: String,
    email: String,
}

impl CsvTable for User {
    fn csv_headers() -> Vec<String> {
        vec!["Name".into(), "Email".into()]
    }

    fn csv_row(&self) -> Vec<String> {
        vec![self.name.clone(), self.email.clone()]
    }
}
