1//! Hiver Observability - Distributed tracing, metrics, and structured logging
2//! Hiver可观测性 - 分布式追踪、指标和结构化日志
3//!
4//! # Overview / 概述
5//!
6//! `hiver-observability` provides a unified observability stack for the Hiver framework,
7//! encompassing distributed tracing, metrics collection, and structured logging. It follows
8//! a design philosophy of "observable by default" - every request is automatically traced
9//! without manual instrumentation.
10//!
11//! `hiver-observability` 为Hiver框架提供统一的可观测性技术栈，涵盖分布式追踪、指标收集
12//! 和结构化日志。它遵循"默认可观测"的设计理念——每个请求都会被自动追踪，无需手动埋点。
13//!
14//! # Features / 功能
15//!
16//! - **Distributed Tracing** - OpenTelemetry-compatible span propagation and context management /
17//!   分布式追踪 - 兼容OpenTelemetry的span传播和上下文管理
18//! - **Metrics** - Counters, gauges, and histograms with a pluggable registry / 指标 -
19//!   计数器、仪表盘和直方图，支持可插拔注册表
20//! - **Structured Logging** - JSON/text output with log levels, rotation, and configurable
21//!   formatting / 结构化日志 - 支持JSON/文本输出，包含日志级别、轮转和可配置格式化
22//! - **Hiver Format** - Branded startup banner and colored console output (feature-gated) /
23//!   Hiver格式 - 品牌化启动横幅和彩色控制台输出（特性门控）
24//! - **Zipkin Export** - Send trace data to Zipkin collectors (feature-gated) / Zipkin导出 -
25//!   将追踪数据发送到Zipkin收集器（特性门控）
26//!
27//! # Equivalent to Spring Boot / 等价于 Spring Boot
28//!
29//! - Spring Boot Actuator `/actuator/metrics` and `/actuator/health` -> `MetricsRegistry`
30//! - Micrometer Tracing (`MicrometerObservation`) -> `Tracer`, `Span`, `TraceContext`
31//! - SLF4J + Logback (`application.yml` logging config) -> `Logger`, `LoggerConfig`
32//! - Spring Boot Startup Banner -> `Banner` (behind `hiver-format` feature)
33//!
34//! # Module Organization / 模块组织
35//!
36//! | Module | Purpose / 用途 |
37//! |--------|---------------|
38//! | [`log`] | Structured logging with levels, rotation, and multiple output formats / 结构化日志，支持级别、轮转和多种输出格式 |
39//! | [`metrics`] | Counters, gauges, histograms, and the central metrics registry / 计数器、仪表盘、直方图及中央指标注册表 |
40//! | [`mod@trace`] | Distributed tracing with spans, trace IDs, and context propagation / 分布式追踪，包含span、追踪ID和上下文传播 |
41//!
42//! # Example / 示例
43//!
44//! ```rust,no_run,ignore
45//! use hiver_observability::{info, LoggerConfig, Logger, MetricsRegistry, Tracer};
46//!
47//! fn main() {
48//!     // Initialize logging / 初始化日志
49//!     let config = LoggerConfig::default();
50//!     let _logger = Logger::new(config);
51//!
52//!     info!("Application started");
53//!
54//!     // Record metrics / 记录指标
55//!     let registry = MetricsRegistry::new();
56//!     let counter = registry.counter("requests_total");
57//!     counter.increment();
58//! }
59//! ```
