1//! hiver-vault — Spring Vault equivalent for `HashiCorp` Vault secrets management
2//! hiver-vault — `HashiCorp` Vault 密钥管理（Spring Vault 等价实现）
3//!
4//! # Overview / 概述
5//!
6//! `hiver-vault` provides a complete Rust client for `HashiCorp` Vault,
7//! equivalent to Spring Vault in the Java/Spring ecosystem.
8//!
9//! `hiver-vault` 提供完整的 `HashiCorp` Vault Rust 客户端，
10//! 等价于 Java/Spring 生态中的 Spring Vault。
11//!
12//! # Features / 功能
13//!
14//! - **`VaultClient`** — HTTP client with TLS and token auth / 支持 TLS 和 Token 认证的 HTTP 客户端
15//! - **Auth** — Token & `AppRole` authentication / Token 和 `AppRole` 认证
16//! - **Secret** — CRUD operations on secrets / 密钥的增删改查操作
17//! - **Lease** — Lease renewal and revocation / 租约续订和撤销
18//! - **KV** — KV v1 and v2 secret backends / KV v1 和 v2 密钥后端
19//! - **Transit** — Encrypt/decrypt as a service / 加密/解密服务
20//! - **PKI** — Certificate management / 证书管理
21//! - **Health** — Health check and seal status / 健康检查和封印状态
22//!
23//! # Quick Start / 快速开始
24//!
25//! ```rust,no_run,ignore
26//! use hiver_vault::{VaultClient, VaultConfig, auth::TokenAuth};
27//!
28//! #[tokio::main]
29//! async fn main() -> Result<(), Box<dyn std::error::Error>> {
30//!     let config = VaultConfig::builder()
31//!         .address("https://127.0.0.1:8200")
32//!         .token("my-root-token")
33//!         .build()?;
34//!
35//!     let client = VaultClient::connect(config)?;
36//!
37//!     // Read a secret / 读取密钥
38//!     let secret = client.kv_v2("secret")
39//!         .read("myapp/config")
40//!         .await?;
41//!
42//!     Ok(())
43//! }
44//! ```
