Condition expression evaluator for event filtering 用于事件过滤的条件表达式求值器

Provides a simple (non-SpEL) expression language for filtering events before they reach listeners. Supports property-based comparisons with logical combinators (And, Or, Not).

提供简单的（非SpEL）表达式语言，用于在事件到达监听器之前进行过滤。 支持基于属性的比较和逻辑组合器（And、Or、Not）。

§Supported Syntax / 支持的语法

expression    := or_expr
or_expr       := and_expr ("or" and_expr)*
and_expr      := not_expr ("and" not_expr)*
not_expr      := "not" not_expr | primary
primary       := property_comparison | "(" expression ")"
property_comparison := path operator value
path          := identifier ("." identifier)*
operator      := "==" | "!=" | ">" | "<" | "contains"
value         := string_literal | number_literal
string_literal := "'" [^']* "'"
number_literal := [0-9]+ ("." [0-9]+)?
