Item writer trait 项目写入器trait

Writes items in chunks (batches) for efficient batch processing. 以块（批次）写入项目以实现高效的批处理。

§Spring Equivalent / Spring等价物

public interface ItemWriter<T> {
    void write(Chunk<? extends T> chunk) throws Exception;
}

§Example / 示例

ⓘ
use hiver_batch::prelude::*;
use async_trait::async_trait;

struct DatabaseWriter {
    pool: sqlx::PgPool,
}

#[async_trait]
impl ItemWriter for DatabaseWriter {
    type Item = User;

    async fn write(&mut self, items: Vec<User>) -> BatchResult<()> {
        let mut tx = self.pool.begin().await?;
        for user in items {
            sqlx::query("INSERT INTO users (name, email) VALUES ($1, $2)")
                .bind(&user.name)
                .bind(&user.email)
                .execute(&mut *tx)
                .await?;
        }
        tx.commit().await?;
        Ok(())
    }
}
