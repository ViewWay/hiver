# hiver-data-orm

**ORM integration for the Hiver Framework (Active Record, Query Builder).**

**Hiver框架的 ORM 集成（Active Record、Query Builder）。**

## Overview / 概述

`hiver-data-orm` provides ORM abstractions including Active Record pattern, Query Builder, model derive macros, relationships, and migrations.

`hiver-data-orm` 提供 ORM 抽象，包括 Active Record 模式、Query Builder、Model 派生宏、关联关系和数据库迁移。

## Features / 功能

See [API Documentation](https://docs.rs/hiver-data-orm) for details.

详见 [API 文档](https://docs.rs/hiver-data-orm)。

## Installation / 安装

```toml
[dependencies]
hiver-data-orm = { version = "0.1" }
```
