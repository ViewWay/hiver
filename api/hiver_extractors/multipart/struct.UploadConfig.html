Expand description

Configuration for file upload limits and constraints. 文件上传限制和约束的配置。

Equivalent to Spring’s MultipartResolver configuration properties. 等价于 Spring 的 MultipartResolver 配置属性。

§Example / 示例

ⓘ
use hiver_extractors::multipart::UploadConfig;

let config = UploadConfig::builder()
    .max_file_size(5 * 1024 * 1024)   // 5MB per file
    .max_total_size(50 * 1024 * 1024)  // 50MB total
    .max_files_per_field(5)
    .allowed_extensions(vec!["jpg", "png", "gif".to_string()])
    .allowed_mime_types(vec!["image/".to_string()])
    .build();

Fields§

§max_file_size: usize

Maximum size of a single file in bytes. Default: 10MB. 单个文件的最大大小（字节）。默认：10MB。

§max_total_size: usize

Maximum total size of all files in a request in bytes. Default: 100MB. 请求中所有文件的最大总大小（字节）。默认：100MB。

§max_files_per_field: usize

Maximum number of files per form field. Default: 10. 每个表单字段的最大文件数量。默认：10。

§allowed_extensions: Vec<String>

Allowed file extensions (lowercase, without dot). Empty = allow all. 允许的文件扩展名（小写，不带点）。空 = 允许所有。

§allowed_mime_types: Vec<String>

Allowed MIME type prefixes. Empty = allow all. 允许的 MIME 类型前缀。空 = 允许所有。

§upload_dir: String

Directory for saving uploaded files. Default: “./uploads”. 保存上传文件的目录。默认：“./uploads”。
