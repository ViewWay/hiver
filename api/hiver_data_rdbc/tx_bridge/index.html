Module tx_bridge 

Source
Expand description

Transaction bridge between hiver-data-rdbc and hiver-tx. hiver-data-rdbc 与 hiver-tx 之间的桥接事务模块。

Provides bidirectional IsolationLevel conversions and a RdbcTransactionManager that implements hiver_tx::TransactionManager by delegating to a DatabaseClient.

提供双向 IsolationLevel 转换和 RdbcTransactionManager， 通过委托给 DatabaseClient 来实现 hiver_tx::TransactionManager。

§Feature / 特性

This module is only available when the tx-bridge feature is enabled: 此模块仅在启用 tx-bridge feature 时可用：

[dependencies]
hiver-data-rdbc = { version = "0.1", features = ["tx-bridge"] }
