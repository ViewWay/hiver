MongoDB Index Management MongoDB 索引管理

§Overview / 概述

Provides index creation, deletion, and listing functionality. Equivalent to Spring Data MongoDB’s @Indexed, @CompoundIndex, and IndexOperations. 提供索引的创建、删除和列表功能。 等价于 Spring Data MongoDB 的 @Indexed、@CompoundIndex 和 IndexOperations。

§Example / 示例

ⓘ
use hiver_data_mongodb::index::{IndexBuilder, IndexDirection};
use mongodb::bson::doc;

let index = IndexBuilder::new()
    .field("email", IndexDirection::Ascending)
    .unique()
    .sparse()
    .build();
