Expand description

Rate-limiting strategy. Each algorithm (token bucket, sliding window, fixed window) implements this trait so the RateLimiter holds a single boxed strategy instead of three parallel Option<Arc<State>> fields dispatched by match. This is the Strategy pattern from GoF. 限流策略。每种算法（令牌桶、滑动窗口、固定窗口）实现此 trait， 使 RateLimiter 持有单个装箱策略，而非三个并行 Option<Arc<State>> 字段再用 match 分派。这是 GoF 策略模式。
