Expand description

Executes the full bean lifecycle for a given bean instance. 为给定的Bean实例执行完整的生命周期。

This is a convenience function that applies the standard lifecycle:

  1. Before-initialization post-processors
  2. InitializingBean::after_properties_set (if implemented)
  3. After-initialization post-processors

这是一个便捷函数，应用标准生命周期：

  1. 初始化前的后处理器
  2. InitializingBean::after_properties_set（如果已实现）
  3. 初始化后的后处理器

§Arguments / 参数

  • bean — The bean instance to initialize. 要初始化的Bean实例。
  • bean_name — The name of the bean. Bean的名称。
  • processors — Slice of BeanPostProcessors to apply. 要应用的 BeanPostProcessor 切片。
