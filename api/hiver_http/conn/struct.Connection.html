Struct Connection

pub struct Connection { /* private fields */ }
Expand description

Represents an active HTTP connection 表示活动的HTTP连接

§Example / 示例

ⓘ
use hiver_http::conn::Connection;
use std::net::SocketAddr;

let addr: SocketAddr = "127.0.0.1:8080".parse().unwrap();
let conn = Connection::with_address(addr);

assert_eq!(conn.remote_addr(), Some("127.0.0.1:8080"));
assert!(conn.is_alive());
assert!(conn.state() == ConnectionState::Active);

pub fn is_alive(&self) -> bool

Check if the connection is still alive 检查连接是否仍然活动

A connection is considered alive if: 连接被视为活动需要满足：

  • It has not been closed
  • 它未被关闭
  • It has not exceeded the maximum idle time
  • 它未超过最大空闲时间

pub fn close(&mut self) -> Result<()>

Close the connection 关闭连接

Marks the connection as closed. Returns an error if already closed. 将连接标记为已关闭。如果已关闭则返回错误。

§Example / 示例
ⓘ
use hiver_http::conn::Connection;

let mut conn = Connection::new();
assert!(conn.is_alive());

conn.close()?;
assert!(!conn.is_alive());
assert!(matches!(conn.state(), ConnectionState::Closed));

pub fn force_close(&mut self)

Force close the connection without checking current state 强制关闭连接而不检查当前状态

This will set the connection to closed state regardless of current state. 这将把连接设置为关闭状态，无论当前状态如何。
