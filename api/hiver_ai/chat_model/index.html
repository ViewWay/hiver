Chat model abstractions for AI/LLM interactions. AI/LLM 交互的聊天模型抽象。

This module defines the core types for interacting with chat-based AI models: messages, requests, responses, streaming, and token usage tracking.

本模块定义了与基于聊天的 AI 模型交互的核心类型： 消息、请求、响应、流式传输和 token 使用量跟踪。
