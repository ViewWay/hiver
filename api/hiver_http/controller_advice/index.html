Global Exception Handler — equivalent to Spring’s @ControllerAdvice 全局异常处理器 – 等价于 Spring 的 @ControllerAdvice

Provides a centralized mechanism for handling errors across all HTTP handlers, similar to how Spring’s @ControllerAdvice + @ExceptionHandler works.

§Example / 示例

ⓘ
use hiver_http::controller_advice::*;

let advice = ControllerAdvice::builder()
    .handler(NotFoundHandler::new())
    .handler(ValidationHandler::new())
    .build();

let response = advice.handle(&error::Error::not_found("User"));
// Returns 404 JSON error response
