Struct DefaultConversionService

pub struct DefaultConversionService { /* private fields */ }
Expand description

Default conversion service with built-in converters. 带内置转换器的默认转换服务。

Supports common conversions: 支持常见转换：

  • String → i8, i16, i32, i64, i128, isize
  • String → u8, u16, u32, u64, u128, usize
  • String → f32, f64
  • String → bool
  • String → Duration (e.g., “5s”, “100ms”, “1h30m”)
