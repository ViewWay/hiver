# hiver-retry

**Retry logic for the Hiver Framework (equivalent to Spring Retry).**

**Hiver框架的重试逻辑（等价于 Spring Retry）。**

## Overview / 概述

`hiver-retry` provides configurable retry logic with exponential backoff, retry policies, and circuit breaker integration.

`hiver-retry` 提供可配置的重试逻辑，支持指数退避、重试策略和熔断器集成。

## Features / 功能

See [API Documentation](https://docs.rs/hiver-retry) for details.

详见 [API 文档](https://docs.rs/hiver-retry)。

## Installation / 安装

```toml
[dependencies]
hiver-retry = { version = "0.1" }
```
