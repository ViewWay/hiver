# hiver-async

**Async task execution framework for the Hiver Framework (equivalent to Spring @Async and TaskExecutor).**

**Hiver框架的异步任务执行框架（等价于 Spring @Async 和 TaskExecutor）。**

## Overview / 概述

`hiver-async` provides async task execution abstractions, including task executors, scheduled tasks, and future composition.

`hiver-async` 提供异步任务执行抽象，包括任务执行器、定时任务和 Future 组合。

## Features / 功能

See [API Documentation](https://docs.rs/hiver-async) for details.

详见 [API 文档](https://docs.rs/hiver-async)。

## Installation / 安装

```toml
[dependencies]
hiver-async = { version = "0.1" }
```
