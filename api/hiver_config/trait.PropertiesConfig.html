Properties configuration trait 属性配置trait

Equivalent to Spring Boot’s @ConfigurationProperties. 等价于Spring Boot的@ConfigurationProperties。

This trait should be derived using the PropertiesConfig derive macro. 此trait应该使用PropertiesConfig派生宏来派生。
