# hiver-i18n

**Internationalization (i18n) support for the Hiver Framework.**

**Hiver框架的国际化（i18n）支持。**

## Overview / 概述

`hiver-i18n` provides internationalization support with message sources, locale resolution, and resource bundle management.

`hiver-i18n` 提供国际化支持，支持消息源、区域设置解析和资源包管理。

## Features / 功能

See [API Documentation](https://docs.rs/hiver-i18n) for details.

详见 [API 文档](https://docs.rs/hiver-i18n)。

## Installation / 安装

```toml
[dependencies]
hiver-i18n = { version = "0.1" }
```
