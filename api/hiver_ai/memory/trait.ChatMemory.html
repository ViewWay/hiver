Expand description

Trait for chat memory implementations that store conversation history. 用于存储对话历史的聊天记忆实现的 trait。

Implementations can back conversation storage with in-memory data structures, databases, or external services.

实现可以使用内存数据结构、数据库或外部服务作为对话存储后端。
