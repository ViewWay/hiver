hiver_data_commons::part_tree

Struct PartTree 

Source 
pub struct PartTree { /* private fields */ }
Expand description

The fully parsed PartTree from a method name. 从方法名完全解析的 PartTree。

This is the main entry point for the PartTree parser. It holds the subject, keywords, part tree, and order-by clauses. 这是 PartTree 解析器的主入口点。 它保存主体、关键字、部件树和排序子句。

§Example / 示例

use hiver_data_commons::part_tree::PartTree;

let tree =
    PartTree::parse("findByNameAndAgeGreaterThanOrderByNameDesc").expect("valid method name");

assert_eq!(tree.subject().to_string(), "FIND");
assert!(!tree.is_distinct());
assert_eq!(tree.parts().len(), 2);
assert_eq!(tree.order_by().len(), 1);
