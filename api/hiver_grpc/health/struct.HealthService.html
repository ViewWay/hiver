Struct HealthService 

Source 
pub struct HealthService { /* private fields */ }
Expand description

In-memory health check service for tracking serving status of gRPC services. 基于 gRPC 服务内存的健康状态跟踪器。

§Example / 示例

ⓘ
use hiver_grpc::health::{HealthService, ServingStatus};

let health = HealthService::new();
health.set_serving("my.package.MyService");
assert_eq!(health.check("my.package.MyService"), ServingStatus::Serving);
