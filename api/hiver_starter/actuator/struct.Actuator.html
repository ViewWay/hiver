§Spring Equivalent / Spring 等价物

Equivalent to Spring Boot Actuator with endpoints: /health, /info, /metrics

§Example / 示例

ⓘ
use hiver_actuator::Actuator;

let actuator = Actuator::new()
    .info("my-app", "1.0.0")
    .enable_health(true)
    .enable_metrics(true);
