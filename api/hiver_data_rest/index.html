Hiver Data REST — Spring Data REST equivalent. Hiver Data REST — Spring Data REST 等价功能。

Auto-generates type-safe REST CRUD handlers from PagingAndSortingRepository traits. No reflection, no runtime scanning — pure Rust generics.

从 PagingAndSortingRepository trait 自动生成类型安全的 REST CRUD 处理器。 无反射，无运行时扫描 — 纯 Rust 泛型。

§Spring Equivalent / Spring等价物

.get(id)GET /resource/{id} (item)

§Rust Advantage / Rust优势

  • Compile-time: wrong type = compile error (Spring: runtime 500)
  • No reflection: generic dispatch, zero overhead
  • Type-safe IDs: ID: FromStr + Display, not raw strings
