1//! # Hiver MCP — Model Context Protocol
2//!
3//! Production-grade MCP server & client for the Hiver framework.
4//! Implements MCP specification version `2025-03-26` over JSON-RPC 2.0.
5//!
6//! Hiver 框架的模型上下文协议(MCP)实现，支持 `2025-03-26` 版本规范。
7//!
8//! # Design Patterns / 设计模式
9//!
10//! | Pattern       | Application                                    |
11//! |---------------|------------------------------------------------|
12//! | Strategy      | `Transport` trait — stdio / HTTP swappable     |
13//! | State         | `ServerLifecycle` — state-machine lifecycle    |
14//! | Builder       | `McpServerBuilder` / client config             |
15//! | Registry      | Separate registries for tools/resources/prompts|
16//! | Adapter       | `HiverToolAdapter` ↔ hiver-ai `ToolCallback`  |
17//! | Command       | Method dispatch on incoming JSON-RPC requests  |
18//! | Observer      | Notification callbacks for list changes        |
19//!
20//! # Quick Start — MCP Server / 快速开始 — MCP 服务器
21//!
22//! ```rust,ignore
23//! use hiver_mcp::{McpServer, StdioTransport, Tool, CallToolResult};
24//!
25//! let server = McpServer::builder("my-server", "1.0.0")
26//!     .tool("echo", "Echo input", echo_handler)
27//!     .build();
28//!
29//! let transport = StdioTransport::new();
30//! server.run(transport).await?;
31//! ```
32//!
33//! # Quick Start — MCP Client / 快速开始 — MCP 客户端
34//!
35//! ```rust,ignore
36//! use hiver_mcp::{McpClient, ChildProcessTransport};
37//!
38//! let transport = ChildProcessTransport::new("npx", &["-y", "@anthropic/mcp-server"]);
39//! let client = McpClient::connect(transport).await?;
40//! let tools = client.list_tools().await?;
41//! ```
