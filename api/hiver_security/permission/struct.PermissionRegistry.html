Struct PermissionRegistry 

Source 
pub struct PermissionRegistry { /* private fields */ }
Expand description

Permission registry for dynamic permission management. 动态权限管理的权限注册表。

Stores permission definitions and role-to-permission mappings. 存储权限定义和角色到权限的映射。
