1//! Hiver Batch Framework
2//! Hiver批处理框架
3//!
4//! A batch processing framework equivalent to Spring Batch.
5//! 等价于Spring Batch的批处理框架。
6//!
7//! # Overview / 概述
8//!
9//! This module provides a comprehensive batch processing framework with:
10//! 本模块提供完整的批处理框架，具有：
11//!
12//! - **Job**: Batch job container containing multiple steps **Job**: 包含多个步骤的批处理作业容器
13//! - **Step**: Independent phase of execution within a job **Step**: 作业中独立的执行阶段
14//! - **ItemReader**: Reads items for processing **ItemReader**: 读取待处理项目
15//! - **ItemProcessor**: Transforms items (optional) **ItemProcessor**: 转换项目（可选）
16//! - **ItemWriter**: Writes processed items **ItemWriter**: 写入处理后的项目
17//! - **JobRepository**: Stores job metadata and execution history **JobRepository**:
18//!   存储作业元数据和执行历史
19//! - **JobLauncher**: Launches and manages job execution **JobLauncher**: 启动和管理作业执行
20//!
21//! # Spring Equivalent / Spring等价物
22//!
23//! ```java
24//! // Spring Batch Job Configuration
25//! @Bean
26//! public Job importUserJob(JobRepository jobRepository, Step step1) {
27//!     return new JobBuilder("importUserJob", jobRepository)
28//!         .start(step1)
29//!         .build();
30//! }
31//!
32//! @Bean
33//! public Step step1(JobRepository jobRepository,
34//!                   PlatformTransactionManager transactionManager,
35//!                   ItemReader<User> reader,
36//!                   ItemWriter<User> writer) {
37//!     return new StepBuilder("step1", jobRepository)
38//!         .<User, User>chunk(100)
39//!         .reader(reader)
40//!         .writer(writer)
41//!         .transactionManager(transactionManager)
42//!         .build();
43//! }
44//! ```
45//!
46//! # Example / 示例
47//!
48//! ```rust,ignore
49//! use hiver_batch::prelude::*;
50//! use async_trait::async_trait;
51//!
52//! // Define reader
53//! struct UserReader;
54//!
55//! #[async_trait]
56//! impl ItemReader for UserReader {
57//!     type Item = User;
58//!
59//!     async fn read(&mut self) -> BatchResult<Option<User>> {
60//!         // Read user from source
61//!         Ok(Some(User::default()))
62//!     }
63//! }
64//!
65//! // Define writer
66//! struct UserWriter;
67//!
68//! #[async_trait]
69//! impl ItemWriter for UserWriter {
70//!     type Item = User;
71//!
72//!     async fn write(&mut self, items: Vec<User>) -> BatchResult<()> {
73//!         // Write users to destination
74//!         Ok(())
75//!     }
76//! }
77//!
78//! // Create and run job
79//! #[tokio::main]
80//! async fn main() -> BatchResult<()> {
81//!     let job = Job::new("import-users")
82//!         .add_step(Step::new("process-users")
83//!             .with_chunk_size(100)
84//!             .with_reader(UserReader)
85//!             .with_writer(UserWriter))
86//!         .build()?;
87//!
88//!     let launcher = JobLauncher::new(JobRepository::new());
89//!     let execution = launcher.run(job).await?;
90//!
91//!     println!("Job completed with status: {:?}", execution.status);
92//!     Ok(())
93//! }
94//! ```
