Expand description

Trait for formatting response data based on content negotiation. 内容协商响应格式化 trait。

Equivalent to Spring’s HttpMessageConverter — converts a response object to the requested content type. 等价于 Spring 的 HttpMessageConverter，将响应对象转换为请求的内容类型。

Format the given data into an HTTP response, using the Accept header value to determine the best content type.

将给定数据格式化为 HTTP 响应，使用 Accept header 值确定最佳内容类型。
