Expand description

Unified API response structure 统一 API 响应结构

This is equivalent to Spring’s ResponseEntity<T> or a custom Result<T> wrapper. 这等价于 Spring 的 ResponseEntity<T> 或自定义 Result<T> 包装器。

§JSON Format / JSON 格式

Success response / 成功响应:

{
  "code": 200,
  "message": "Success",
  "data": { "id": 1, "name": "Alice" },
  "timestamp": "2024-01-29T10:30:45Z"
}

Error response / 错误响应:

{
  "code": 404,
  "message": "Not Found",
  "error": "USER_NOT_FOUND",
  "timestamp": "2024-01-29T10:30:45Z"
}
