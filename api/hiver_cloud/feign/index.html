Feign declarative HTTP client — runtime support types. Feign 声明式 HTTP 客户端 — 运行时支持类型。

§Description / 描述

Provides the runtime scaffolding used by the #[feign_client] proc-macro. 提供 #[feign_client] proc-macro 使用的运行时脚手架。

§Example / 示例

ⓘ
use hiver_macros::{feign_client, feign_get, feign_post};
use hiver_cloud::feign::FeignResult;
use serde::{Deserialize, Serialize};

#[derive(Debug, Deserialize)]
pub struct User { pub id: u64, pub name: String }

#[feign_client(url = "http://user-service")]
pub trait UserClient {
    #[feign_get("/users/{id}")]
    async fn get_user(&self, id: u64) -> FeignResult<User>;
}
