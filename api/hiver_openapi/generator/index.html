OpenAPI 3.0 Spec Generator OpenAPI 3.0 规范生成器

Provides builders for generating complete OpenAPI 3.0 specifications programmatically, including schema inference from Rust types. 提供用于编程式生成完整 OpenAPI 3.0 规范的构建器，包括从 Rust 类型推断模式。

§Example / 示例

ⓘ
use hiver_openapi::generator::{OpenApiSpec, SchemaGenerator};

let spec = OpenApiSpec::builder()
    .title("My API")
    .version("1.0.0")
    .build();
let json = spec.to_json().unwrap();
