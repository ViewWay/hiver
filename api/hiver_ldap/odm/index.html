Object-Directory Mapping (ODM) — annotation-driven LDAP mapping 对象目录映射 (ODM) — 注解驱动的LDAP映射

Equivalent to Spring LDAP ODM. Provides utilities to map between Rust structs and LDAP entries.

等价于 Spring LDAP ODM。 提供在Rust结构体和LDAP条目之间映射的工具。
