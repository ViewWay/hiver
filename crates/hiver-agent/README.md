# hiver-agent

**AI Agent framework for the Hiver Framework.**

**Hiver框架的 AI Agent 框架。**

## Overview / 概述

`hiver-agent` provides an AI agent abstraction layer for building intelligent agents with tool use, memory, and planning capabilities.

`hiver-agent` 提供 AI Agent 抽象层，用于构建具有工具调用、记忆和规划能力的智能代理。

## Features / 功能

See [API Documentation](https://docs.rs/hiver-agent) for details.

详见 [API 文档](https://docs.rs/hiver-agent)。

## Installation / 安装

```toml
[dependencies]
hiver-agent = { version = "0.1" }
```
