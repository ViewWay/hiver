OpenAPI Schema OpenAPI模式

Equivalent to Spring’s @Schema annotation. 等价于Spring的@Schema注解。

§Spring Equivalent / Spring等价物

@Schema(description = "User object", example = "{\"id\": 1}")
public class User {
    @Schema(description = "User ID", required = true)
    private Long id;
}
