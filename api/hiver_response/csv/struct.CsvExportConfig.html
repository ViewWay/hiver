Expand description

CSV export configuration. CSV 导出配置。

Equivalent to Spring’s OpenCSV CSVWriter settings. 等价于 Spring 的 OpenCSV CSVWriter 设置。

Fields§

§headers: Vec<String>

Column headers / 列标题

§delimiter: char

Field delimiter (default: comma) / 字段分隔符（默认：逗号）

§quote_char: char

Quote character (default: double-quote) / 引号字符（默认：双引号）

§line_terminator: String

Line terminator / 行终止符

§always_quote: bool

Always quote fields / 总是引用字段

§include_bom: bool

Include BOM for Excel compatibility / 包含 BOM 以兼容 Excel
