# hiver-data-commons

**Common data access abstractions for the Hiver Framework.**

**Hiver框架的通用数据访问抽象。**

## Overview / 概述

`hiver-data-commons` provides common traits and types for data access, including Repository patterns, Page/Sort, entity metadata, and `MethodName::parse()` for findByXxxAndYyy.

`hiver-data-commons` 提供数据访问的通用 trait 和类型，包括 Repository 模式、Page/Sort、实体元数据，以及 `MethodName::parse()` 用于 findByXxxAndYyy。

## Features / 功能

See [API Documentation](https://docs.rs/hiver-data-commons) for details.

详见 [API 文档](https://docs.rs/hiver-data-commons)。

## Installation / 安装

```toml
[dependencies]
hiver-data-commons = { version = "0.1" }
```
