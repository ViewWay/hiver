§Overview / 概述

This module provides a condition evaluation system for the IoC container, enabling Spring Boot-style conditional bean registration. Beans can be registered only when certain conditions are met, such as the presence of a configuration property, the absence of another bean, or an active profile.

此模块为IoC容器提供条件评估系统，支持Spring Boot风格的条件化Bean注册。 只有在满足特定条件时（如配置属性存在、另一个Bean不存在、或特定Profile激活）， Bean才会被注册。
