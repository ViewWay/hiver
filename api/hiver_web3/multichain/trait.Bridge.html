Expand description

Cross-chain bridge interface. 跨链桥接口。

Defines the interface for cross-chain token bridge operations. Individual bridge implementations (e.g. across chain, Stargate) should implement this trait.

定义跨链代币桥接操作的接口。 各个桥接实现（例如 across chain、Stargate）应实现此trait。
