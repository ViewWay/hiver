组件扫描器 / Component Scanner

自动发现和注册应用中的组件。 Automatically discover and register components in the application.

参考 Spring Boot 的 @ComponentScan。 Based on Spring Boot’s @ComponentScan.

In Hiver, component discovery is performed at compile time via inventory::submit! (from #[service], #[component], #[bean], #[configuration] macros). The scanner introspects these registered beans rather than scanning source files.

在 Hiver 中，组件发现通过 inventory::submit! 在编译时完成 （来自 #[service]、#[component]、#[bean]、#[configuration] 宏）。 扫描器检查这些已注册的 Bean，而非扫描源文件。
