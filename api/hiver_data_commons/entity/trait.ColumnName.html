Column mapping trait 列映射 trait

Maps entity fields to database column names. 将实体字段映射到数据库列名。

§Example / 示例

ⓘ
use hiver_data_commons::ColumnName;

struct User {
    id: i32,
    user_name: String,
}

impl ColumnName for User {
    fn column_name(field: &str) -> String {
        match field {
            "id" => "id".to_string(),
            "user_name" => "username".to_string(),
            _ => field.to_string(),
        }
    }
}
