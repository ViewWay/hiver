Hiver Macros - Spring Boot Style Procedural Macros Hiver宏 - Spring Boot风格的过程宏

§Overview / 概述

hiver-macros provides Spring Boot-style procedural macros for the Hiver framework.

hiver-macros 为 Hiver 框架提供 Spring Boot 风格的过程宏。

§Example / 示例

ⓘ
use hiver_macros::{hiver_main, controller, get};

#[hiver_main]
struct Application;

fn main() -> anyhow::Result<()> {
    Application::run()
}

#[controller]
struct DemoController;

#[get("/helloworld")]
fn hello() -> &'static str {
    "Hello World!"
}
