OpenAI embedding model client. OpenAI 嵌入模型客户端。

Implements the EmbeddingModel trait for generating vector embeddings using the OpenAI Embeddings API.

实现 EmbeddingModel trait，使用 OpenAI 嵌入 API 生成向量嵌入。
