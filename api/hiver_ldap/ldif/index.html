LDIF (LDAP Data Interchange Format) Parser LDIF（LDAP 数据交换格式）解析器

Parses LDIF content as defined in RFC 2849. Supports add, modify, delete, and moddn change types with base64-encoded values and URL references. 解析 RFC 2849 中定义的 LDIF 内容。支持 add、modify、delete 和 moddn 更改类型，包括 base64 编码值和 URL 引用。

§Example / 示例

ⓘ
use hiver_ldap::ldif::{parse_ldif, LdifEntry, LdifChangeType};

let input = r#"
dn: cn=John Doe,dc=example,dc=com
changetype: add
objectClass: person
cn: John Doe
sn: Doe
"#;

let entries = parse_ldif(input).unwrap();
assert_eq!(entries.len(), 1);
assert_eq!(entries[0].dn, "cn=John Doe,dc=example,dc=com");
