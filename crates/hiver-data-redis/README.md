# hiver-data-redis

**Redis integration for the Hiver Framework (RedisTemplate, Cache).**

**Hiver框架的 Redis 集成（RedisTemplate、Cache）。**

## Overview / 概述

`hiver-data-redis` provides Redis integration with RedisTemplate, caching abstractions, and pub/sub support.

`hiver-data-redis` 提供 Redis 集成，支持 RedisTemplate、缓存抽象和发布/订阅。

## Features / 功能

See [API Documentation](https://docs.rs/hiver-data-redis) for details.

详见 [API 文档](https://docs.rs/hiver-data-redis)。

## Installation / 安装

```toml
[dependencies]
hiver-data-redis = { version = "0.1" }
```
