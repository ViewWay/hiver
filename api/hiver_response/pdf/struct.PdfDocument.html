Expand description

A PDF document containing one or more pages. 包含一页或多页的 PDF 文档。

Equivalent to Spring’s JasperReports JasperPrint or iText PdfDocument. 等价于 Spring 的 JasperReports JasperPrint 或 iText PdfDocument。
