# hiver-retry-macros

**Procedural macros for the Hiver retry system.**

**Hiver重试系统的过程宏。**

## Overview / 概述

`hiver-retry-macros` provides the `#[retry]` procedural macro for declarative retry configuration.

`hiver-retry-macros` 提供 `#[retry]` 过程宏，用于声明式重试配置。

## Features / 功能

See [API Documentation](https://docs.rs/hiver-retry-macros) for details.

详见 [API 文档](https://docs.rs/hiver-retry-macros)。

## Installation / 安装

```toml
[dependencies]
hiver-retry-macros = { version = "0.1" }
```
