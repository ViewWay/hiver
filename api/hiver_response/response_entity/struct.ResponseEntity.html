Struct ResponseEntity 

Source 
pub struct ResponseEntity<T> { /* private fields */ }
Expand description

Spring-style generic response entity with status, headers, and body. Spring 风格的泛型响应实体，包含状态码、头部和请求体。

§Equivalent to Spring / 等价于 Spring

// Spring Java
ResponseEntity.ok(user)
ResponseEntity.status(201).body(newUser)
ResponseEntity.notFound().build()
ⓘ
// Hiver Rust
use hiver_response::ResponseEntity;

ResponseEntity::ok(user)
ResponseEntity::created("/users/1").body(newUser)
ResponseEntity::not_found().build()
