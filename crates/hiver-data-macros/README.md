# hiver-data-macros

**Procedural macros for the Hiver data layer.**

**Hiver数据层的过程宏。**

## Overview / 概述

`hiver-data-macros` provides procedural macros for deriving Repository implementations, entity mappings, and query generation.

`hiver-data-macros` 提供过程宏，用于自动派生 Repository 实现、实体映射和查询生成。

## Features / 功能

See [API Documentation](https://docs.rs/hiver-data-macros) for details.

详见 [API 文档](https://docs.rs/hiver-data-macros)。

## Installation / 安装

```toml
[dependencies]
hiver-data-macros = { version = "0.1" }
```
