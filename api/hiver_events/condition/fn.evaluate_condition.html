pub fn evaluate_condition<E: ConditionPropertyProvider + Any>(
    condition: &dyn EventCondition,
    event: &E,
) -> bool
Expand description

Evaluate a condition against a typed event that implements ConditionPropertyProvider. 对实现了 ConditionPropertyProvider 的类型化事件求值条件。

For PropertyCondition and CompositeCondition containing property conditions, this delegates to matches_provider. For other condition types, it falls back to EventCondition::matches.

对于 PropertyCondition 和包含属性条件的 CompositeCondition， 此函数委托给 matches_provider。对于其他条件类型，回退到 EventCondition::matches。
