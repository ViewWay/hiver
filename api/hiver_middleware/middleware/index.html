§Overview / 概述

This module re-exports the Middleware trait and Next type from hiver-router. All middleware implementations should use these types for compatibility.

此模块从hiver-router重新导出Middleware trait和Next类型。 所有中间件实现应使用这些类型以确保兼容性。
