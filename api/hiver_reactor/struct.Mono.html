pub struct Mono<T> { /* private fields */ }
Expand description

Mono<T> — an asynchronous publisher of zero or one T value, followed by completion or an error.

Mono<T>——零或一个 T 值的异步发布者，随后是完成或错误。

Backed by a boxed Future<Output = ReactorResult<Option<T>>>. None represents an empty completion (the Reactor Mono.empty() case).

由 boxed Future<Output = ReactorResult<Option<T>>> 支撑。None 表示空完成 （Reactor Mono.empty() 的情况）。

§Spring Equivalent / Spring 等价物

Mono<T> ↔ Project Reactor Mono<T>.

§Example / 示例

use hiver_reactor::Mono;
let v = Mono::just(7).map(|x| x + 1).await.unwrap();
assert_eq!(v, Some(8));
