Specification API for type-safe dynamic queries 用于类型安全动态查询的 Specification API

§Overview / 概述

This module provides a JPA-inspired Specification pattern for building dynamic queries in a composable, type-safe manner. 本模块提供 JPA 风格的 Specification 模式，以可组合、类型安全的方式构建动态查询。

§Example / 示例

ⓘ
use hiver_data_commons::specification::{Specifications, SpecValue};

let spec = Specifications::where_clause()
    .eq("status", SpecValue::String("active".into()))
    .gt("age", SpecValue::I64(18))
    .build();
