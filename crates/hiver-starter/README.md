# hiver-starter

**Spring Boot-style auto-configuration for the Hiver Framework.**

**Hiver框架的 Spring Boot 风格自动配置。**

## Overview / 概述

`hiver-starter` provides auto-configuration and dependency aggregation, combining all Hiver modules into a single dependency.

`hiver-starter` 提供自动配置和依赖聚合，将所有 Hiver 模块组合为单一依赖。

## Features / 功能

See [API Documentation](https://docs.rs/hiver-starter) for details.

详见 [API 文档](https://docs.rs/hiver-starter)。

## Installation / 安装

```toml
[dependencies]
hiver-starter = { version = "0.1" }
```
