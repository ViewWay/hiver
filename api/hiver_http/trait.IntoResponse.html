Expand description

Trait for types that can be converted to HTTP responses 可转换为HTTP响应的类型trait

This is equivalent to Spring’s ResponseEntity or methods annotated with @ResponseBody.

这等价于Spring的ResponseEntity或使用@ResponseBody注解的方法。
