Struct RdbcTransactionManager 

Source 
pub struct RdbcTransactionManager<C> { /* private fields */ }
Expand description

A TransactionManager implementation that delegates to a DatabaseClient. 通过委托给 DatabaseClient 实现的 TransactionManager。

This allows hiver-tx’s TransactionTemplate and #[Transactional] to work with any database backend supported by hiver-data-rdbc.

使 hiver-tx 的 TransactionTemplate 和 #[Transactional] 能够与 hiver-data-rdbc 支持的任何数据库后端一起工作。

§Example / 示例

ⓘ
use hiver_data_rdbc::{DatabaseClient, tx_bridge::RdbcTransactionManager};
use hiver_tx::TransactionTemplate;

let client = SqlxPoolClient::new(pool).await?;
let tx_mgr = RdbcTransactionManager::new(client, "default");
let template = TransactionTemplate::new(tx_mgr);

let result = template.execute(|| async {
    // Database operations here run within the rdbc transaction
    Ok(())
}).await?;
