Redis Distributed Lock — mutex, reentrant lock, and watchdog renewal. Redis 分布式锁 — 互斥锁、可重入锁和看门狗续期。

§Overview / 概述

Three lock variants equivalent to Redisson’s lock primitives:

TypeRedisson equivalentDescription
RedisLockgetLock()Non-reentrant mutex (SET NX EX)
ReentrantRedisLockRLock (reentrant)Reentrant via Lua HASH counter
WatchdogGuardRedisson watchdogAuto-renews TTL in background

§Example — reentrant lock with watchdog / 示例 — 带看门狗的可重入锁

ⓘ
use hiver_data_redis::lock::ReentrantRedisLock;

let rlock = ReentrantRedisLock::new(client, "order:42", 30);
let guard = rlock.acquire_with_watchdog("worker-1", None).await?.unwrap();
// guard auto-renews every 10 s; watchdog stops on drop
guard.release().await?;
