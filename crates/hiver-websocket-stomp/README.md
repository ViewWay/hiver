# hiver-websocket-stomp

**STOMP protocol support over WebSocket for the Hiver Framework.**

**Hiver框架的 WebSocket STOMP 协议支持。**

## Overview / 概述

`hiver-websocket-stomp` provides STOMP protocol support over WebSocket for real-time messaging with pub/sub semantics.

`hiver-websocket-stomp` 提供基于 WebSocket 的 STOMP 协议支持，用于具有发布/订阅语义的实时消息传递。

## Features / 功能

See [API Documentation](https://docs.rs/hiver-websocket-stomp) for details.

详见 [API 文档](https://docs.rs/hiver-websocket-stomp)。

## Installation / 安装

```toml
[dependencies]
hiver-websocket-stomp = { version = "0.1" }
```
