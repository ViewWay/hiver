# hiver-flyway

**Database migration support for the Hiver Framework (equivalent to Flyway).**

**Hiver框架的数据库迁移支持（等价于 Flyway）。**

## Overview / 概述

`hiver-flyway` provides database migration management with versioned SQL scripts, rollback support, and migration history tracking.

`hiver-flyway` 提供数据库迁移管理，支持版本化 SQL 脚本、回滚和迁移历史追踪。

## Features / 功能

See [API Documentation](https://docs.rs/hiver-flyway) for details.

详见 [API 文档](https://docs.rs/hiver-flyway)。

## Installation / 安装

```toml
[dependencies]
hiver-flyway = { version = "0.1" }
```
