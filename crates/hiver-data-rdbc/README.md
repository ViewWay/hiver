# hiver-data-rdbc

**Reactive database access for the Hiver Framework (R2DBC equivalent).**

**Hiver框架的响应式数据库访问（R2DBC 等价物）。**

## Overview / 概述

`hiver-data-rdbc` provides reactive database connectivity with connection pooling, RowMapper, and ResultSetExtractor abstractions.

`hiver-data-rdbc` 提供响应式数据库连接，支持连接池、RowMapper 和 ResultSetExtractor 抽象。

## Features / 功能

See [API Documentation](https://docs.rs/hiver-data-rdbc) for details.

详见 [API 文档](https://docs.rs/hiver-data-rdbc)。

## Installation / 安装

```toml
[dependencies]
hiver-data-rdbc = { version = "0.1" }
```
