Hiver Resilience - High availability patterns Hiver弹性 - 高可用模式

§Overview / 概述

hiver-resilience provides high availability patterns such as circuit breakers, rate limiters, retry policies, and service discovery.

hiver-resilience 提供高可用模式，如熔断器、限流器、重试策略和服务发现。
