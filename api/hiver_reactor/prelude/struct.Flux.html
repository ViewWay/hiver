Struct Flux

pub struct Flux<T> { /* private fields */ }
Expand description

Flux<T> — an asynchronous publisher of zero or more T values, followed by completion or an error.

Flux<T>——零或多个 T 值的异步发布者，随后是完成或错误。

It is a thin newtype wrapper over Stream<Item = ReactorResult<T>> plus operator methods. Cold sources are constructed lazily and re-evaluated per subscription; hot sources come from Sinks.

它是对 Stream<Item = ReactorResult<T>> 的轻量 newtype 包装，外加算子方法。 冷源惰性构造、每次订阅重新求值；热源来自 Sinks。
