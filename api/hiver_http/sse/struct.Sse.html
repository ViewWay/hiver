SSE response builder SSE响应构建器

Creates an HTTP response with SSE headers and event data. 创建带有SSE头部和事件数据的HTTP响应。

§Example / 示例

ⓘ
use hiver_http::sse::{Event, Sse};

// Create SSE response with a single event
let response = Sse::new()
    .with_event(Event::data("Hello, World!"))
    .into_response();

// With custom headers
let response = Sse::new()
    .with_retry(3000)
    .with_last_event_id("123")
    .with_event(Event::data("Update").id("456"))
    .into_response();
