A received HTTP response with fluent assertion methods.

接收到的HTTP响应，带有流式断言方法。

§Spring Equivalent / Spring等价物

webTestClient.get().uri("/api/users/1")
    .exchange()
    .expectStatus().isOk()
    .expectHeader().contentType(MediaType.APPLICATION_JSON)
    .expectBody(User.class);
