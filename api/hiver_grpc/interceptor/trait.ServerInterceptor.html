Expand description

Server-side gRPC interceptor. 服务端 gRPC 拦截器。

Implement this trait to add cross-cutting logic (auth, logging, tracing) to every incoming RPC call. 实现此 trait 为每个传入 RPC 调用添加横切逻辑（认证、日志、追踪）。

Intercept an incoming request. 拦截传入请求。

Return Ok(request) to continue the call chain, or Err(Status) to reject. 返回 Ok(request) 继续调用链，返回 Err(Status) 以拒绝请求。
