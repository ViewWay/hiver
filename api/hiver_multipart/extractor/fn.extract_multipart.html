Extract Multipart from request 从请求提取 Multipart

This is typically used internally by the framework. 这通常由框架内部使用。

§Parameters / 参数

  • req - The HTTP request to extract multipart data from 从中提取多部分数据的 HTTP 请求
  • config - Configuration for multipart processing 多部分处理的配置

§Returns / 返回

Returns Ok(Multipart) if the request contains valid multipart/form-data, or an error if the content type is missing/invalid or the body cannot be read.

如果请求包含有效的 multipart/form-data，则返回 Ok(Multipart)， 如果内容类型缺失/无效或无法读取主体，则返回错误。
