Postman Collection v2.1 generator Postman Collection v2.1 生成器

Converts OpenAPI 3.0 routes to Postman Collection v2.1 format. 将 OpenAPI 3.0 路由转换为 Postman Collection v2.1 格式。

§Example / 示例

ⓘ
use hiver_openapi::postman::PostmanGenerator;
use hiver_openapi::OpenApiBuilder;

let openapi = OpenApiBuilder::new()
    .title("My API")
    .version("1.0.0")
    .add_path("/users", /* ... */)
    .build();

let collection = PostmanGenerator::from_openapi(&openapi, "http://localhost:8080");
let json = collection.to_json().unwrap();
