AOP procedural macros for Hiver Framework. Hiver框架的AOP过程宏。

Provides #[Aspect], #[Before], #[After], #[Around], #[AfterReturning], #[AfterThrowing], #[Pointcut].

Usually imported via hiver_aop re-exports rather than directly.

Attribute Macros§

after
Marks a method as after advice. 将方法标记为后置通知。

Marks a method as after-returning advice. 将方法标记为返回后通知。

Marks a method as after-throwing advice. 将方法标记为异常后通知。
around
Marks a method as around advice. 将方法标记为环绕通知。
