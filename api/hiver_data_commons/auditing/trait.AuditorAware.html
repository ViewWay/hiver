Expand description

Trait for providing the current auditor. 提供当前审计者的 trait。

Implementations resolve the current user/principal from the security context or other authentication mechanism. 实现从安全上下文或其他身份验证机制解析当前用户/主体。

This mirrors Spring Data’s AuditorAware<T> interface. 这镜像了 Spring Data 的 AuditorAware<T> 接口。

§Example / 示例

use hiver_data_commons::auditing::AuditorAware;

struct SecurityAuditor;

impl AuditorAware<String> for SecurityAuditor
{
    fn current_auditor(&self) -> Option<String>
    {
        // In a real app, read from security context
        // 在实际应用中，从安全上下文读取
        Some("admin".to_string())
    }
}

fn current_auditor(&self) -> Option<T>

Returns the current auditor. 返回当前审计者。

Returns None if no auditor is available (e.g., anonymous access). 如果没有可用的审计者（如匿名访问），返回 None。
