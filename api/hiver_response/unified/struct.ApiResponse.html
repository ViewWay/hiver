Struct ApiResponse

pub struct ApiResponse<T: Serialize> {
    pub code: u16,
    pub message: String,
    pub data: Option<T>,
    pub timestamp: u64,
}
Expand description

Unified API response wrapper. 统一 API 响应包装器。

All API responses are automatically wrapped in this structure:

{"code": 200, "message": "success", "data": {...}, "timestamp": 1234567890}

Fields§

§code: u16

Business status code. 200 for success, 4xx/5xx for errors. 业务状态码。200 表示成功，4xx/5xx 表示错误。

§message: String

Human-readable message. 可读消息。

§data: Option<T>

Response payload, None for error responses. 响应数据，错误响应时为 None。

§timestamp: u64

Unix timestamp (seconds). Unix 时间戳（秒）。
