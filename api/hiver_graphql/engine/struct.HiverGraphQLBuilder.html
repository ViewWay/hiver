Expand description

Convenience builder for constructing a Hiver-wrapped async-graphql schema. 方便构建 Hiver 包装的 async-graphql schema 的辅助构建器。

§Example / 示例

ⓘ
let engine = HiverGraphQLBuilder::new(QueryRoot, EmptyMutation, EmptySubscription)
    .max_depth(10)
    .introspection(true)
    .build();
