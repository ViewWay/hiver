Reactive streams — Mono and Flux 响应式流抽象 — Mono 和 Flux

§Overview / 概述

Provides Mono<T> (0..1 item) and Flux<T> (0..N items) abstractions, equivalent to Spring WebFlux’s reactive types.

提供 Mono<T>（0..1 个元素）和 Flux<T>（0..N 个元素）抽象， 等价于 Spring WebFlux 的响应式类型。

§Example / 示例

use hiver_core::reactive::{Flux, Mono};

async fn example()
{
    // Mono — single optional value / 单个可选值
    let mono = Mono::just(42);
    let val = mono.block().await; // Some(42)

    // Flux — stream of values / 值流
    let flux = Flux::from_iter(vec![1, 2, 3]);
    let items = flux.collect::<Vec<_>>().await;
}
