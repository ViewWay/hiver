Trait for chat model implementations. 聊天模型实现的 trait。

This trait defines the interface for all chat-based AI models, enabling pluggable backends (OpenAI, Anthropic, local models, etc.).

此 trait 定义了所有基于聊天的 AI 模型的接口， 支持可插拔的后端（OpenAI、Anthropic、本地模型等）。
