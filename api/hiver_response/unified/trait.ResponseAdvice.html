Expand description

Response advice trait for automatic wrapping. 自动包装的响应通知 trait。

Implement this trait to provide custom wrapping logic. The default implementation simply wraps data into ApiResponse::success. The framework can call ResponseAdvice::wrap in middleware to automatically envelope handler return values.

实现此 trait 以提供自定义包装逻辑。默认实现将数据包装到 ApiResponse::success。框架可以在中间件中调用 ResponseAdvice::wrap 来自动包装处理器返回值。
