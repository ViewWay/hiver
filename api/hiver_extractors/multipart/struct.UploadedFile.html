Expand description

Represents an uploaded file received in a multipart request. 表示在 multipart 请求中接收的上传文件。

Equivalent to Spring’s MultipartFile. 等价于 Spring 的 MultipartFile。

§Example / 示例

ⓘ
use hiver_extractors::multipart::UploadedFile;

// Save uploaded file to disk
file.save_to("./uploads/photo.jpg")?;

// Check file type
if file.is_image() {
    println!("Image: {} ({} bytes)", file.original_name, file.size);
}
